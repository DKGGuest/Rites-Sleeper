# SARTHI - Rites Sleeper Application

Automated Real-time Tracking & Health Inspection (SARTHI) for sleeper production.

## Modules
- **Sleeper Process Duty**: SCADA synchronization, manual quality checks, moisture analysis.
- **Raw Material Inspection**: Quality testing for cement, aggregates, water, etc.
- **Call Desk Dashboard**: Monitoring and dispatch.

## Tech Stack
- React 19
- Vite 7
- Vanilla CSS

## Development
```bash
npm install
npm run dev
```
